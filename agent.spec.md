# Build a Polished Landing Page — Agent Specification

## What This Project Builds

A complete marketing landing page for a product or service. Students build every section
from scratch and ship something they would not be embarrassed to show a potential customer.

## Stack

Next.js 15 (App Router) · TypeScript · Tailwind CSS v4

## Student Prompt

Copy and paste this into your AI coding tool:

---

Build a polished marketing landing page using Next.js 15 App Router with TypeScript and Tailwind CSS v4.

The page must have these five sections in order:

1. **Hero** — Full-width section with a bold headline (max 8 words), a supporting subheadline (one sentence), and a primary CTA button ("Get Started Free") + a secondary ghost button ("See How It Works"). Include a subtle gradient background.

2. **Features Grid** — A 3-column grid (collapses to 1 on mobile). Each card has an icon, a short title, and a 2-sentence description. Include six features total.

3. **Social Proof** — Three testimonial cards in a row. Each card shows a quote, the person's name, role, and company. Use a subtle card border and a star rating (5 stars).

4. **Pricing** — Two pricing tiers side by side: Free (list 4 features, "Get Started" CTA) and Pro at $19/month (list 6 features, "Start Free Trial" CTA, visually highlighted with a border and "Most Popular" badge).

5. **Footer** — Company name, tagline, three columns of navigation links (Product, Company, Legal), and a copyright line.

Design requirements:
- Light background with dark text
- Consistent spacing between sections (at least `py-24`)
- All text content should use realistic but clearly placeholder copy
- Mobile responsive — all grids collapse cleanly on small screens
- Sticky navbar with logo on the left, nav links + CTA button on the right
