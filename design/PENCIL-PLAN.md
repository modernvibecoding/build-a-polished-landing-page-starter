# Landing Page Layout Specification

This document describes the intended layout and structure for each section of the landing page. Use it as a reference when building and to check whether your sections match the intended design.

**Theme:** Light mode — white or near-white backgrounds, dark text, one accent color
**Canvas width:** 1280px desktop

---

## Section Layout Reference

### Desktop Layout

1. **Sticky Navbar** — Logo left, nav links center, CTA button right. Height ~64px. White background with subtle bottom border.

2. **Hero Section** — Full-width. Bold headline (large text, centered or left-aligned). Subheadline below. Two buttons side by side (primary filled, secondary ghost/outline). Subtle gradient or soft color background. Generous padding top and bottom.

3. **Features Grid** — 3-column grid. 6 cards total (2 rows of 3). Each card shows an icon placeholder, title, and 2-line description. Even spacing between cards. Section heading above the grid.

4. **Social Proof** — 3 testimonial cards in a row. Each card has a quote block, name, role/company line, and 5-star rating. Subtle card borders or shadows.

5. **Pricing** — 2 pricing cards side by side. Left card (Free) is plain. Right card (Pro) has an accent border and "Most Popular" badge. Each card has a title, price, feature list, and CTA button.

6. **Footer** — Dark or muted background. Company name + tagline. 3 columns of links. Copyright line at bottom.

### Mobile Layout

The same 6 sections stacked vertically at 375px width. Features grid collapses to single column. Pricing cards stack. Testimonials stack. Navbar shows hamburger icon or simplified layout.

---

## Visual Style Notes

- Use realistic placeholder text — not "Lorem ipsum"
- One consistent accent color throughout (blue, indigo, or green work well)
- Clear vertical spacing between sections — this is the primary teaching point
- Buttons: filled primary, outline secondary
- No decorative illustrations — keep it structural
- Font: sans-serif (Inter or equivalent)
