# Plan — Build a Polished Landing Page

## Project Goal

Design and build a conversion-focused landing page that looks production-ready: strong hero, clear value proposition, social proof, pricing or feature sections, and a call-to-action — all with professional typography, spacing, and responsive layout.

## Todo List

- [ ] Write a one-sentence product description
- [ ] Write the hero headline (8 words max) and subheadline
- [ ] Define 6 features to highlight (icon/emoji + 3–5 word title + 2-sentence description each)
- [ ] Write 3 testimonials (quote, name, role, company) for the social proof section
- [ ] Define Free tier (4 features) and Pro tier pricing ($19/mo, 6 features)
- [ ] Decide on a CTA action (sign up, waitlist, buy, learn more)

### Structure
- [ ] Build `HeroSection` component

- [ ] Build `SocialProofSection` (3 testimonial cards: quote, name, role, company, 5-star rating)

### Design
- [ ] Define color palette (primary, accent, background, text)
- [ ] Apply consistent type scale (hero, h2, body, caption)
- [ ] Consistent section padding (`py-20` / `py-24` rhythm)
- [ ] Buttons have hover states and focus rings

### Responsiveness
- [ ] Hero stacks cleanly on mobile
- [ ] Feature grid collapses to single column on small screens
- [ ] Navigation accessible on mobile (hamburger or simplified)

### Ship
